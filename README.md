# Web Assembly for RAGE:MP client

### What is this ?

This is a boilerplate for writing wasm with rust for RAGE:MP client

### Requirement:

-   Nodejs installed
-   Rustup installed

### How to use ?

1. npm install
2. npm run build
3. read [here](https://rustwasm.github.io/wasm-bindgen/) the basic usage of this...
